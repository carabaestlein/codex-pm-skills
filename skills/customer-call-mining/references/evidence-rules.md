# Evidence Rules

Use evidence compactly. The output should be useful to a PM, not a transcript rewrite.

## Evidence Should Include

- Transcript or call reference
- Account or segment when known
- Speaker role when known
- Short paraphrase or short quote when helpful

## Avoid

- Long quotes
- Unsupported deal impact
- Unsupported urgency
- Treating salesperson interpretation as customer evidence
- Combining unrelated issues into one insight

## Source Labels

Use stable, short source labels:

- `Call 1`
- `Gong: Acme renewal - 2026-05-12`
- `Transcript: enterprise-admin-feedback.md`

If metadata is missing, say `Unknown` rather than guessing.
