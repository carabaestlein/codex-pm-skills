# Output Format

Use this section order:

1. Theme Map
2. Use Cases
3. Requirements
4. Product Feedback
5. Open Questions, only if needed
6. Source Coverage

For one transcript, use `Emphasis`. For multiple transcripts, use `Frequency`.

## Theme Map

```markdown
## Theme Map

| Theme | Summary | Evidence | Segment / Persona | Emphasis/Frequency | Impact | Confidence |
|---|---|---|---|---|---|---|
| <Theme> | <What we heard> | <Call refs or short evidence> | <ICP/persona/account type> | <High/Medium/Low or # calls> | <High/Medium/Low> | <High/Medium/Low> |
```

## Use Cases

```markdown
## Use Cases

| Use Case | Persona | Current Workflow | Desired Outcome | Evidence | Impact | Confidence |
|---|---|---|---|---|---|---|
| <Use case> | <User/persona> | <How they do it today> | <What they want to achieve> | <Call refs> | <High/Medium/Low> | <High/Medium/Low> |
```

## Requirements

```markdown
## Requirements

| Requirement | Need It Serves | Deal / Retention Relevance | Segment | Urgency | Evidence | Impact | Confidence |
|---|---|---|---|---|---|---|---|
| <Requirement> | <Underlying need> | <Blocker / Accelerator / Nice-to-have / Churn risk / Unknown> | <ICP/persona/account type> | <Now / This quarter / Later / Unknown> | <Call refs> | <High/Medium/Low> | <High/Medium/Low> |
```

## Product Feedback

```markdown
## Product Feedback

| Feedback | Type | Product Area | Evidence | Segment / Persona | Impact | Confidence |
|---|---|---|---|---|---|---|
| <Concrete feedback> | <Bug / Gap / Enhancement / UX friction / Pricing-packaging / Integration / Other> | <Product area> | <Call refs or short evidence> | <ICP/persona> | <High/Medium/Low> | <High/Medium/Low> |
```

## Open Questions

Use only when needed:

```markdown
## Open Questions

- <Question needed to interpret the evidence or decide next steps>
```

## Source Coverage

```markdown
## Source Coverage

| Source | Account / Segment | Call Type | Date | Participants | Notes |
|---|---|---|---|---|---|
| <Transcript or call title> | <Account/segment if known> | <Sales / CS / Research / Support / Unknown> | <Date or Unknown> | <Roles if known> | <Caveats> |
```
